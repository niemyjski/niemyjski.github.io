Microsoft Visual Studio 2008 SP1 and .NET Framework 3.5 SP1 RTM

December 28, 2008

2 minute read

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Email

I was so happy to see that Microsoft has finally release Visual Studio 2008 SP1 and the .NET Framework 3.5 SP1. It contains so many improvements and bug fixes; you can view the full list of the enhancements at Scott Guthrie’s blog. “Unlike other Service Packs from Microsoft, the SP1 of.Net Framework 3.5 and Visual Studio 2008 not only contains bug fixes and improvements, it also contains new technologies such as ADO.Net Entity Framework v1, ADO.Net Data Services v1, ASP.Net Dynamic Data and the framework and tools support for working with SQL Server 2008 that was RTM’ed last week.
