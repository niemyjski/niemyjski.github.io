Microsoft Releases Visual Studio 2005 Service Pack 1

November 21, 2007

1 minute read

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Email

Today, Microsoft released Visual Studio 2005 Service Pack 1 available for download immediately.

Service Pack 1 also provides over 70 improvements for common development scenarios including:

  • New processor support (e.g., Core Duo) for code generation and profiling
  • Performance and scale improvements in Team Foundation Server*
  • Team Foundation Server integration with Excel 2007 and Project 2007*
  • Tool support for occasionally connected devices and SQL Server Compact Edition*
  • Additional support for project file based Web applications*
  • Windows Embedded 6.0 platform and tools support
