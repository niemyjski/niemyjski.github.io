Windows Server 2008 is way faster than a Leopard..

January 19, 2008

1 minute read

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Email

Last night my Windows Vista x64 Development box crashed and burned. This all came about after I rebooted my machine after a good days use and was greeted with a BSOD. After googling the words wudfrd which appeared in the BSOD, I found out that the ITunes update I installed earlier in the day was the root of the problem. Thanks Apple for taking the time to modify my system files for me.

I decided that instead of formatting with Vista, I would try Release Candidate One of Windows Server 2008 x64 Enterprise. WOW, I can honestly say that I was shocked by how much faster my applications installed and ran. I’ve never used an operating system that was so quick all around. I beg anyone with a 64bit CPU to go and experience it, you won’t be let down!

I’ll unquestionably be finding myself reformatting again later this week when Server 2008 has RTMed!
