Service Pack 1 released for .NET 2.0 and 3.0

November 21, 2007

1 minute read

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Email

Yesterday Microsoft complimented the release of Visual Studio 2008 with Service Pack 1 for both .NET 2.0 and 3.0. Information and download links are provided below.

“Microsoft .NET Framework 2.0 Service Pack 1 provides cumulative roll-up updates for customer reported issues found after the release of Microsoft .NET Framework 2.0. In addition, this release provides security improvements, and prerequisite feature support for .NET Framework 3.0 Service Pack 1, and .NET Framework 3.5* (Microsoft).
