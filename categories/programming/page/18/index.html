January 22, 2008

1 minute read

The Microsoft .NET Micro Framework 2.0 SP1 is the efficient runtime that powers today’s most popular SideShow-compatible devices. The Windows SideShow Device SDK enables developers to build SideShow-compatible devices using the .NET Micro Framework 2.0 SP1.
